LowkeySpots

Some places are not on google maps, and sometimes you want to keep it that way!

LowkeySpots is an interactive social platform that provides users the ability to mark locations from the real world onto maps which are privately shared between users. Users have the ability to add friends, and then add those friends to their maps as either spectators or contributors. Furthermore, users may add images and designate colors to their marked locations. The app is designed using Django Rest Framework, and ReactJS. It utilizes a PostgreSQL server and is deployed on Google Cloud Run.
